Funcionamento geral do sistema: o sistema funcionara a base de instruções simples como por exemplo pedir para um robo fazer um café: 1. pegue o pó do café, filtro e uma chaleira com água 2(seria o processamento). coloque o filtro sobre o suporte, coloque o pó do café no filtro, esquente a água que esta na chaleira, coloque a água quente no pó de café 3(seria o fim). café está pronto
Explicação das decisões aplicadas no algoritimo: as decisões aplicadas no algoritimo foram escolhidas para ter Clareza, cada passo deve ser simples o suficiente para ser entendido, ordem, a sequência importa (não dá para enxaguar a boca antes de colocar a pasta), fim, ele precisa chegar a um resultado final.
